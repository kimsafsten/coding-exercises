## SafstensBank console application 

A bank app where the user log in using their social security number, if the number is new the user states their name.
The user can withdraw and deposit money on their acount, see the balance of the account and all transactions mmade on the acount.
The bank stores all data in a json file and testpersons are loaded at the first run of the program.
