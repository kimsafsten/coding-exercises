### Blackjack

Detta projekt är ett Blackjack-spel i C#, utvecklat som en konsolapplikation.
Uppdraget var att bygga ett fungerande spel utan att använda objekt eller klasser — all logik hanteras därför med metoder, listor och variabler i stället för objektorienterad programmering.

Spelet följer de klassiska reglerna för Blackjack:
spelaren börjar med 200 kr, satsar valfritt antal 5-kronor och försöker komma så nära 21 som möjligt utan att gå över. Dealern drar kort tills summan är minst 17.
Spelet avgör automatiskt vinnare, hanterar insatser och låter spelaren välja om hen vill spela vidare eller avsluta.
